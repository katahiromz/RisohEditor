RisohEditor від katahiromz

RisohEditor — це безкоштовний редактор ресурсів для розробки під Win32.
Він може редагувати/видобувати/клонувати/видаляти дані ресурсів у файлах 
RC/RES/EXE/DLL.

Посилання на завантаження: https://katahiromz.web.fc2.com/re/en

Працює на Windows XP/2003/Vista/7/8.1/10 та ReactOS.

Див. файл «LICENSE.txt» для отримання детальної інформації про авторські 
права та ліцензійну угоду.

/////////////////////////////////////////////////////

Запитання 1. Що таке "Risoh"?

    Відповідь. Слово "Risoh" означає "ідеальний" японською мовою.

Запитання 2. Що таке edt1, edt2, cmb1?

    Відповідь. Це стандартні макроси ідентифікатора керування, визначені в
               <dlgs.h>.

Запитання 3. Що таке mcdx?

    Відповідь. Це спеціальний компілятор повідомлень, який я створив.
               Детальніше див. mcdx/MESSAGETABLEDX.md.

Запитання 4. Чому під час компіляції за допомогою Visual Studio з'являлися
             спотворені символи?

    Відповідь. rc.exe коректно підтримує UTF-16, але до версії
               Visual Studio 2022 завантаження файлу UTF-8 призводило до появи
               сміття у вихідних даних.

               Будь ласка, використовуйте Visual Studio 2022 або пізнішу
               версію.

Запитання 5. Яка різниця між версією без інсталятора та портативною версією?

    Відповідь. Портативна версія використовує не реєстр, а INI-файл.

Запитання 6. Чи підтримує RisohEditor 64-бітну версію?

    Відповідь. Так, підтримує.
